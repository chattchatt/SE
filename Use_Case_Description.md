
## <허수민 팀원>
### Use case description : 자전거 등록
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
|1. 관리자가 자전거 등록 창에 접근한다.
|2. 자전거 정보(자전거 ID, 자전거 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중) 등)를 입력할 수 있는 화면을 보여준다.|
|3. 관리자가 자전거 정보(자전거 ID, 자전거 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중) 등)를 입력하고 등록 버튼을 클릭한다.
|4. 입력된 자전거 정보를 확인하고 저장한다.|
|5. None
|6. 정상적으로 저장되었음을 알려준다.|


### Use case description : 자전거 리스트 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
|1. 관리자가 자전거 리스트 조회 창에 접근한다.
|2. 등록된 자전거 목록을 보여준다.|
|Extension
A. 관리자가 목록 내 특정 자전거 삭제 버튼을 클릭하는 경우 해당 자전거 항목을 삭제할 수 있다.|


### Use case description : 자전거 상세 내용 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
|1. 관리자가 자전거 목록 화면에서 특정 자전거를 선택한다.
|2. 해당 자전거 항목의 상세 내용(자전거 ID, 자전거 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중) 등)을 보여준다.|


### Use case description : 로그인
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
|1. 이용자가 로그인 창에 접근한다.
|2. ID와 비밀번호를 입력할 수 있는 화면을 보여준다.|
|3. 이용자가 ID와 비밀번호를 입력한다.
|4. ID와 비밀번호를 확인한다.|
|5. None
|6. 메인화면을 보여준다.|


### Use case description : 로그아웃
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
|1. 이용자가 로그아웃 버튼을 클릭한다.
|2. 시스템 접속을 종료하고 초기 화면을 표시한다.|

## <전채운 팀원>
### Use case description : 자전거 대여 정보 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
| ‘내 대여 정보’ 메뉴를 선택한다.        |현재 대여 중인 자전거 목록을 표시한다 (대여소 이름, 대여소 위치, 자전거 ID, 자전거 제품명, 자전거 유형 포함)|
| 특정 자전거 항목을 선택한다.          |선택한 자전거의 상세 정보를 표시한다.|

### Use case description : 대여 금액 및 횟수 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
| ‘통계’ 메뉴를 선택한다.        |전체 대여 정보와 함께 기간별 조회 옵션(1주일/1개월/1년)을 표시한다.|
| 원하는 기간을 선택한다.          |선택한 기간의 총 대여 금액과 대여 횟수를 표시한다.|

## <유현모 팀원>
### Use case description : 대여소 검색
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
| 1. 회원이 대여소 검색 메뉴로 접근한다.        |2. 검색 입력창을 표시한다.|
| 3. 회원이 대여소 이름을 입력한다.   		 |4. 조건에 맞는 대여소 목록을 표시한다.|

### Use case description : 대여소 상세 정보 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
| 1. 회원이 대여소 목록에서 대여소를 선택한다.       |2. 대여소 상세 정보(위치, 자전거 목록)를 표시한다.|

### Use case description : 대여 자전거 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
| 1. 회원이 대여 정보를 조회한다.         |2. 대여 중인 자전거의 상세 정보를 표시한다.|

### Use case description : 예약 대기 조회
| Actor Action | System Response                                                                          |
|--------------|------------------------------------------------------------------------------------------|
| 1. 회원이 예약대기 목록을 조회한다.        |2. 예약대기 정보(대여소 이름, 위치, 자전거 ID, 제품명, 유형 등)를 표시한다.|
